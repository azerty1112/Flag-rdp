منصة لإنشاء جلسات RDP متتالية مع مزامنة المجلدات في حساب GitHub الخاص بالمستخدم.
- إنشاء تطبيق OAuth في GitHub.
- إضافة PAT في `config.php`.
- إنشاء مستودع قالب عام `rdp-template-sync` وإضافة workflow أعلاه.
- إضافة `NGROK_TOKEN` و `UPLOAD_SECRET` (وهو `md5(SECRET_KEY)`) إلى المستودع القالب.
1. المستخدم يسجل دخوله.
2. يرفع مجلد ZIP.
3. يحدد عدد الجلسات المتتالية.
4. تبدأ الجلسة الأولى، يتم تحميل المجلد على سطح المكتب.
5. بعد التطوير، تنتهي الجلسة تلقائياً ويتم رفع المجلد المحدث.
6. تبدأ الجلسة التالية تلقائياً مع المجلد المحدث.
